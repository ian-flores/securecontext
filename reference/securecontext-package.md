# securecontext: Memory, Knowledge Persistence, and Context Management for R LLM Agents

Provides memory, knowledge persistence, RAG retrieval, and token-aware
context window management for R LLM agents. Includes document chunking,
TF-IDF embeddings, vector search, persistent key-value knowledge stores,
and priority-based context assembly.
